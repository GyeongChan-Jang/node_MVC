# 노드로 서버 구축하기

- express 사용
- MVC 패턴
- 템플릿 엔진 사용(ejs, pug)
- fs, path, body-parser 모듈 사용 연습

## MVC 패턴

### MVC에서의 model의 역할

- 데이터를 나타내고, 관리하고, 저장하고, 불러오고, 업데이트 하는등의 역할을 수행

### MVC에서의 view의 역할

- 사용자에게 보여지는 부분을 담당

### MVC에서의 controller의 역할

- 모델과 뷰를 연결하는 역할
- 사용자의 입력을 받아서, model을 업데이트하고, view를 업데이트하는 역할을 수행
